# 柯里化

# 什么是柯里化

Currying(柯里化)是这么一种机制，它将一个接收多个参数的函数，拆分成多个接收单个参数的函数。 参考一下代码：

function add(x, y) {
  return x + y
}

add(2, 3)  // 5

add接收两个参数x，y，并返回它们的和x + y。 经过curry处理，函数形成如下形式：

function add(x){
  return function (y) {
    return x + y
  }
}

add(2)(3) // 5
let add2 = add(2)
add2(3)  // 5

add接收一个参数x，返回另一个接收一个参数y的函数.

# 偏函数

区别于Currying，如果在拆分入参的过程中，这些拆分出来的函数不是一次只应用其中的一个，而是任意多个，则这些函数就是部分应用(Partial application)了原函数中的入参，称作偏函数。 参考如下函数：

function sum(x, y, z) {
  return x + y + z
}

如下的函数就都是偏函数，它们都部分应用了sum的入参：

function partial1(x) {
  return function(y, z) {
    return x + y + z
  }
}
function partial2(x, y) {
  return function(z) {
    return x + y + z
  }
}

偏函数中入参的拆分和部分应用，并不仅限于一层的拆分，可以是任意多次的

function partial3(x) { // curry函数
  return function(y){
    return function(z) {
      return x + y + z
    }
  }
}

# 函数组合(compose)

const compose = fn1 => fn2 => args => fn1(fn2(args))

compose还不够一般化，只能接收两个函数并对其进行结合

const pipe = (...fns) => args => fns.reduce((res, fn) => fn(res), args)
